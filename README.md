# Snake

Single file NCurses implementation of the game Snake

Compile like:

gcc Snake.c -lncurses -lm -o Snake

Run: ./Snake

Cool features:

1. The game screen is resizable while playing. Nice to use with TMUX.

2. Penalty apples when timeout is reached
